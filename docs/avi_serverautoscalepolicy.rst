Examples
--------

.. code-block:: yaml

      connection: local

        - name: Example to create ServerAutoScalePolicy object
          avi_serverautoscalepolicy:
            controller: 192.168.15.18
            username: admin
            password: something
            state: present
            name: sample_serverautoscalepolicy
