Examples
--------

.. code-block:: yaml

    - hosts: localhost
      connection: local
      collections:
        - vmware.alb
      tasks:
        - name: Example to create ALBServicesConfig object
          avi_albservicesconfig:
            controller: 192.168.15.18
            username: admin
            password: something
            state: present
            name: sample_albservicesconfig
